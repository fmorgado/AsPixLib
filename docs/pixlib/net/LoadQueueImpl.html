addThenDispose()method 
public function addThenDispose(command:DataCommand, onComplete:Function, onError:Function, onProgress:Function = null):void

Start command, invokes given listeners then disposes the command. When the command completes, onComplete is invoked with the result and the command is disposed. When the command fails, onError is invoked with the error and the command is disposed.

Parameters

command:DataCommand — The command to start and dispose.
 
onComplete:Function — The result handler. The function signature must be function(data:Object):void.
 
onError:Function — The error handler. The function signature must be function(error:Log):void.
 
onProgress:Function (default = null) — The progress handler. The function signature must be function(progress:Number, total:Number):void.
